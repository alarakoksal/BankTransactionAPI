# Bank Transaction Tracker API

Bu proje, hesap oluşturma ve transaction işlemlerini yönetmek için geliştirilmiş basit bir bankacılık REST API uygulamasıdır.

## Özellikler
- Hesap oluşturma
- Hesapları listeleme
- Transaction listeleme
- Transaction filtreleme

## Teknolojiler
- .NET Web API
- Entity Framework Core
- SQLite

## Amaç
Bu proje backend geliştirme ve katmanlı mimariyi öğrenmek amacıyla yapılmıştır.
